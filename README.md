# Internal Tool

React + Vite + Tailwind internal tool with forms, tables, charts, and real-time updates.

**Stack:** Vite | **Complexity:** moderate
**Tags:** `dashboard`, `internal`, `charts`, `tables`, `tailwind`, `admin`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```

## Claude Code
